# PregMath Client

Frontend React para o projeto PregMath.

## Comandos

- `npm install` - instala dependências do client
- `npm run dev` - inicia o servidor de desenvolvimento do Vite
- `npm run build` - cria a versão de produção
- `npm run preview` - pré-visualiza o build de produção

## Observação

A API backend continua no diretório raiz deste repositório. No futuro, o frontend pode se comunicar com ela em `http://localhost:3000`.
